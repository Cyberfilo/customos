# CustomOS docs

Documentation for CustomOS lives here.

- **`architecture.md`** — current understanding of the system
  architecture, the three-subsystem split, and what's still undecided.
- **`interface.md`** — the contract between the profile extractor and
  the customization system. Stub for now; will be filled when the
  extractor's schema migrates into `customos-core`.
- **`decisions/`** — Architecture Decision Records (ADRs). Append-only.
  Each ADR captures a single decision with its context and consequences.
  Start with `decisions/README.md` for the format.
- **`research/`** — raw research dumps from exploration phases. Dated
  and intentionally unpolished. Conclusions drawn from research live in
  `architecture.md` or as ADRs, not here.
